# Question 3

Create dataset

``` r
set.seed(12345)

sim_df = function(mu,n=30,sigma=5) {
  data =rnorm(n=30, mean = mu, sd=sigma)
  p_value=t.test(data, mu=0 ,alpha = 0.05)
  p_value |> 
    broom::tidy() |> 
    select(estimate, p.value)
}

sim_results_df= map(1:5000,\(i) sim_df(mu=0))|> bind_rows()
```
